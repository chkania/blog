chkania.net
===========
personal website and blog


Prerequisites
-------------
* Ruby (only tested with 1.9)
	* sass gem
	* front-compiler gem

Install 
-------
* `mkdir _cache _compiled _site`
* git clone in `_compiled`  -- this pushes to master
